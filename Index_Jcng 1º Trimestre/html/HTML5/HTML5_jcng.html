Head

La etiqueta head puede contener diversa información sobre la página web, indicando al navegador qué tipo de documento es, su titulo, etc.

Body

El elemento body de HTML representa el contenido de un documento HTML. Solo puede haber un elemento body en un documento.

Estilos

Con la etiqueta HTML style podremos definir estilos CSS en la misma página de manera incrustada en la etiqueta.

Texto

Para que un texto se muestre en nuestra página web simplemente lo introduciremos en la etiqueta body. Y si queremos insertar un párrafo escribiremos el texto entre la apertura y el cierre de la etiqueta p.

Imágenes

Para insertar una imagen en la página web usaremos la etiqueta img.

Hipervínculos

También podremos insertar hipervínculos para redirigirnos a otras páginas webs externas o incluso espacios internas de nuestra propia página, se utiliza la etiqueta a junto hrerf="página".

Listas

Las listas son otra opción de HTML5, podemos listar elementos de manera ordenada con la etiqueta ol o listar de manera no ordenada con ul y listar cada elemento con la etiqueta li.

Tablas

En documentos HTML una tabla puede ser considerada, resumidamente, como un grupo de filas donde cada una contiene a un grupo de celdas, es decir, la creación de la tabla va de izquierda a derecha. Para crear una tabla se utilizara la etiqueta table , por cada fila que creemos, utilizaremos la etiqueta tr y por cada celda, podemos utilizar la etiqueta td y crear celdas con encabezado con th.

Formularios

Los formularios se usan principalmente para obtener datos introducidos por el usuario (como por ejemplo para registrarse en nuestra web o enviarnos un mensaje). Para crear un formulario se utilizan principalmente las etiquetas HTML form, input, textarea y select.

Frames e Iframes

Los frames (también llamados marcos) son usados para dividir una página web en varias partes, cargando otra página web en cada una de ellas.
